## Como ejecutar el projecto?
Primer paso instalamos las dependencias.
```bash
npm install
```
... Despues podemos iniciar el servidor json-server
```bash
npm run server
```
... Y ejecutamos la aplicacion en una nueva instancia de la terminal.
```bash
npm start
```
